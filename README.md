# KVdz00 Portfolio V2

Personal portfolio for Muhammad Dzikrul Kahfi, built to present selected private and public work through concise case-study rows and repo-informed visual previews.

## Stack

- React
- TypeScript
- Vite
- CSS
- Phosphor Icons

## Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run typecheck
npm run build
```
